PHPComplex
==========

---

PHP Class for handling Complex numbers

---

Currently provides functions for:

 - addition
 - subtraction
 - multiplication
 - division
    - division by
    - division into

and

 - abs
 - argument
 - conjugate
 - cos
 - cosh
 - acos
 - sin
 - sinh
 - asin
 - sqrt
 - ln
 - log10
 - log2
